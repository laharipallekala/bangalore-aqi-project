# Bangalore AQI Project
Predictive Modeling of AQI and PM2.5 Levels in Bangalore
A Statistical + Machine Learning Approach

## Project Structure
- notebooks/ — Jupyter notebooks for each phase
- processed/ — cleaned master CSV output

## Datasets Used
- Rohanrao: India Air Quality Data (Kaggle) — AQI, PM2.5, PM10 (2015-2020)
- Open-Meteo API — Temperature, humidity, wind speed, rainfall

## Setup
pip install -r requirements.txt

## Running the Project
Open notebooks/Bengaluru_AQI_project.ipynb in Google Colab
Run cells in order from Phase 1 onwards
